Generated by `elara-project-builder` as an Elara UI RPC client application.

Selected features:
- Application kind: ui
- UI client language: c++
- UI template: tabbed-control-panel
- EPA VM Host adapter: yes
- EPA debug RPC target: yes
- UI RPC server address: 127.0.0.1
- UI RPC server port: 18777

Quick start:
1. start the UI head with `./run-ui-head.sh`
2. build the client with `./build.sh`
3. launch the client with `./run-client.sh`
4. use `sudo ./install.sh` to install the client
5. use `sudo ./install.sh --remove` to uninstall it

Notes:
- the generated client expects a running `libElaraUI` RPC head on 127.0.0.1:18777
- `run-ui-head.sh` defaults to `/usr/local/bin/elaraui-server`
- override that with `ELARA_UI_SERVER=/path/to/elaraui-server ./run-ui-head.sh`
- the C++ client uses `ElaraUiDocumentBuilder` and `ElaraUiRpcPeer`
- if the project lives outside this repo, set `ELARA_ROOT=/path/to/elara/build ./build.sh`
